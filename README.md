## Prerequisite
```
add pywin32==306 and pywinpty==2.0.13 to the requirements.txt
```
## Setup Environment - Shell/Terminal
```
mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
